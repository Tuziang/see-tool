# 网页源代码查看 在线工具分享

平时我们想看一个网页到底加载了什么内容，最常见的方法是按 F12 或右键看源码。但很多时候在手机上、平板上，或者只是想快速查一下页面文本和链接，这些方法就不太方便了。

所以我用 **Vue 3** 做了一个网页源代码查看在线工具，打开就能用，不需要安装任何软件。

> 在线工具网址：[https://see-tool.com/webpage-source-viewer](https://see-tool.com/webpage-source-viewer)  
> 工具截图：  
> ![工具截图](工具截图.png)

这个工具适合普通用户做这些事：

- 看网页原始 HTML，确认页面有没有正常返回内容
- 提取页面里的纯文本，快速复制做整理
- 抓取页面中的链接，方便做收集和检查
- 查看表单相关标签，判断页面里有哪些输入项

使用步骤也很简单：

1. 把目标网址粘贴到输入框
2. 按需选择内容模式（全部源码、纯文本、表单、链接）
3. 点击查看，等待结果返回
4. 用复制或下载按钮保存结果

如果你有更细一点的需求，它还支持自定义 User-Agent、Referer、Cookie 和 POST 参数。对普通用户来说，可以当成“进阶选项”，默认设置就能完成大多数查看任务。

我自己日常会用它来做内容校对、链接排查和页面可读性检查。整体体验就是：页面干净、操作直观、结果返回快，临时用一下非常省时间。

如果你也经常需要“快速看源码、提取文本或链接”，这个工具应该会对你有帮助。
